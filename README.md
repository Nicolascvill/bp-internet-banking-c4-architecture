# bp-internet-banking-c4-architecture
Diseño de arquitectura para sistema de banca por internet BP utilizando modelo C4.

The architecture was designed using the C4 Model and includes:

- Context Diagram
- Container Diagram
- Component Diagram

The solution includes:

- Web Banking SPA
- Mobile Banking Application
- API Gateway
- Microservices for customer, accounts, transactions, notifications and onboarding
- OAuth2 / OpenID Connect authentication
- Redis cache for performance optimization
- PostgreSQL database for persistence and auditing
- Integration with Core Banking System
- Integration with Customer Detail System
- External notification providers (Email / SMS / Push)
- Facial recognition service for onboarding

The full architecture description is available in the PDF document.
